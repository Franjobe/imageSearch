IMAGE SEARCH API

Free Code Camp Backend Challenges

INSTRUCTIONS

Complete the following query with the term you want to search, and the results page number you want to look over:

https://image-seeker01.glitch.me/search/..your term../..page number..

IMPORTANT: write a page number is a must!

EXAMPLE:

https://image-seeker01.glitch.me/search/funny cat/10
(return images of funny cats and display page 10 results)

SEARCH HISTORY:

You can also check out what other people has been looking for. Just Hit:

https://image-seeker01.glitch.me/history
